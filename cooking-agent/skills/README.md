# 烹饪助手 Agent — 技能配置文件索引

本目录存放烹饪助手 Agent 的所有 Skill 定义文件。

## 文件说明

| 文件 | 名称 | 优先级 | 触发词 |
|------|------|--------|--------|
| safety.md | 食品安全 | ⭐⭐⭐（最高） | 能不能吃、中毒、过期、相克 |
| recipe.md | 菜谱查询 | ⭐⭐ | 做法、配方、步骤 |
| suggest.md | 食材推荐 | ⭐⭐ | 推荐、做什么、冰箱里有 |
| nutrition.md | 营养计算 | ⭐⭐ | 热量、卡路里、营养 |
| technique.md | 烹饪技法 | ⭐ | 技法、刀工、火候 |

## 添加新 Skill

1. 在本目录创建 `new-skill.md` 文件
2. 参照现有格式编写（包含元数据、角色设定、工具、回答格式、触发词）
3. 重启服务，SkillLoader 会自动加载

## Skill 加载机制

`src/loader.ts` 启动时扫描本目录，所有 `.md` 文件会被解析，
其 `## 角色设定` 和 `## 回答格式` 内容会注入到系统提示词中，
`## 工具` 表中的工具会被注册到 Function Calling 列表。
